## Python 実行環境

### pipenv インストール

```sh
# Mac の場合
brew install pipenv
pipenv install
pipenv --python 3.9.6
```

### 起動・終了

```sh
# 起動
pipenv shell

# 終了: もしくは Ctrl + D
exit
```

#### test ファイルの実行

```sh
cd src

# python test_two_sum.py など
python ファイル名
```
